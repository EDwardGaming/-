# 寒假哈尔滨三天四夜旅游规划（第二版）

## 酒店选择

- **不选择中央大街旁边的**（贵到离谱）
- **选择市区离地铁站近的**，因为地铁可以抵达主要景点，非常方便且不拥堵。
- **预估人均**：90元/晚
